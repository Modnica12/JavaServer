В докере заходим в папку с этими файлами, затем прописываем эти команды:
docker-compose build
docker-compose up

выйти можно каким-нибудь ctrl+z или ctrl+c


========================== Сборка через мавен ============================

Открыть терминал в директории проекта.

Ввести команду [mvn compile].

Ввести команду [mvexec:java -Dexec.mainClass="com.faceless.Application"].
